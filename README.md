# Expense Management System

##  Overview

This is a web-based Expense Management System developed using the Flask framework and SQLite database. The application allows users to manage their daily expenses in a simple and organized way.

---

## Features

* Add new expenses
* View all expenses
* View individual expense details
* Delete expenses
* Client-side validation
* Basic automated testing using pytest

---

##  Technologies Used

* Python (Flask)
* SQLite
* HTML, CSS, Bootstrap
* pytest

---

##  How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Perera012/Expense-Management-System.git
```

2. Navigate to the project folder:

```
cd Expense-Management-System
```

3. Install dependencies:

```
pip install flask pytest
```

4. Run the application:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000/
```

---

##  Running Tests

```
pytest
```
